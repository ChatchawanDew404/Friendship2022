ทำความรู้จักกับ frontend developer

Frontend Developer คืออะไร

Frontend Developer คือ นักพัฒนาโปรแกรมในส่วนฝั่งของด้านหน้าที่มองเห็นได้ทันทีของหน้าเว็บไซต์ หรือ ที่หลายคนเรียกชอบเรียกกันว่า หน้าบ้าน หรือ หน้าเว็บ นั่นเอง โดยผู้ใช้งานสามารถมองเห็น และมีส่วนร่วมในการโต้ตอบร่วมกันภายในเว็บเบราว์เซอร์ได้ ในส่วนของหน้าเว็บไซต์ หรือแอปพลิเคชันบนมือถือ เป็นส่วนที่ผู้ใช้สามารถมองเห็น และโต้ตอบได้โดยตรง ถูกสร้างขึ้นด้วยภาษาทั่วไปที่ต้องรู้ ยกตัวอย่างเช่น HTML CSS JAVASCRIPT เป็นต้น หากใครอยากทำความรู้จักเเบบเบื้องต้นเข้าใจง่าย ผมได้อธิบายไว้ด้านล่างเรียบร้อย รวมถึงได้อธิบายเกี่ยวกับตัวช่วยในการสร้างเว็บไวต์ หรือที่ชาวโปรเเกรมเมอร์ชอบเรียกกันว่า framework เเละ library เเบบเบื้องต้นด้วยเช่นกัน



ยาวไปไม่อ่าน ขอเเบบสั้นๆ

"Frontend คือ ผู้พัฒนาเว็บไซต์โดยใช้ภาษาโปรเเกรมมิ่ง เเละทำให้ให้ผู้ใช้สามารถโต้ตอบกับเว็บไซต์ได้ ผมได้เเนะนำภาษาเเบบเบื้องต้นไว้เเล้ว"

Web Developer Skill Set

Web development & Technology

HTML5

HTML ย่อมาจาก Hypertext Markup Language มันคือภาษาเขียนเว็บไซต์ที่ใช้เพื่อกำหนดโครงสร้างและแสดงผลคำสั่งบน Browser โครงสร้างของ HTML จะเป็นในรูปแบบของ Tag ต่างๆ และ Web Browser จะแปลความของ Tag แต่ละ Tag ออกมาเป็นหน้าตาเว็บไซต์ "สรุปง่ายๆ HTML5 คือโครงกระดูกของเว็บไซต์"

CSS3

CSS ย่อมาจาก Cascading Style Sheet เป็นภาษาที่ใช้ในการทำส่วนของการจัดรูปแบบการแสดงผลเอกสาร HTML โดยที่ CSS กำหนดรูปแบบ ของเว็บ เช่น สีของข้อความ สีพื้นหลัง พื้นที่ เเละอื่นๆให้เว็บไซต์มีสีสันสวยงามเเละง่ายต่อการใช้งานกับ user "สรุปง่ายๆ CSS คือ เคื่องเเต่งการ"

JAVASCRIPT

JavaScript เป็น ภาษาสคริปต์เชิงวัตถุ ซึ่งในการสร้างและพัฒนาเว็บไซต์ (ใช่ร่วมกับ HTML) เพื่อให้เว็บไซต์ของเราดูมีการเคลื่อนไหว สามารถตอบสนองผู้ใช้งานได้มากขึ้น เช่น กดปุ่ม ส่งข้อความ ทำให้เว็บไซต์ของเรามีปฏิสัมพันธ์กับผู้ใช้งานมากขึ้น "สรุปง่ายๆ JS คือ เส้นประสาท"

BOOTSTRAP

Bootstrap คือชุดคำสั่งที่ประกอบด้วย HTML CSS และ Javascript เป็นชุดคำสั่งที่ถูกพัฒนาขึ้นมาเป็นต้นเเบบเเละสามารถ ดึงคำสั่งจากต้นเเบบ มาปรับปรุงใช้งานได้ เราจึงเรียก Bootstrap ว่าเป็น Front-end framework"สรุปง่ายๆ BS คือ ต้นเเบบคำสั่ง"

JQUERY

Jquery คิอ JavaScript Library ซึ่งถูกออกแบบมาเพื่อให้การเขียน JavaScript นั้นมีความสะดวกและง่ายขึ้น จากเดิมที่ต้องเขียนคำสั่งยาวเป็นหางว่าว ก็ทำให้สั้นลงได้ ใช้งานสะดวก ทำให้การเขียนดปรเเกรมนั้นไวยิ่งขึ้น "สรุปง่ายๆ JQUERY คือ การย่อคำสั่งของ Javascript"

REACT

React คือ JavaScript library ที่ใช้สำหรับสร้าง หน้าเว็บ ทำให้เราสามารถเขียนโค้ดในการสร้างเว็บ ที่มีความซับซ้อนแบ่งเป็นส่วนเล็กๆออกจากกันได้ ซึ่งแต่ละส่วนสามารถแยกการทำงานออกจากกันได้อย่างอิสระ เเละสามารถนำโค้ดกลับมาใช้ใหม่ได้ "สรุปง่ายๆ REACT คือ การ Reuse โค้ด"
